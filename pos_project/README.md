# POS Project Addon

A custom Point of Sale addon for Odoo.

## Description

This module extends the functionality of the Point of Sale application with custom features.

## Features

- Custom POS order management
- Extended POS configuration options

## Installation

1. Copy the `pos_project` directory to your Odoo addons folder
2. Update the app list in Odoo
3. Search for "POS Project Addon" and install it

## Usage

After installation, the new features will be available in the Point of Sale menu.
